# How to install

*    `git clone https://github.com/Code-Theft-Auto/tkinter-stuff.git`

*    `cd tkinter-stuff`

*    `python -m venv .venv`

*    `source .venv/bin/activate` (linux/macos)

* `pip install -r requirements.txt`

* `python player.py`
